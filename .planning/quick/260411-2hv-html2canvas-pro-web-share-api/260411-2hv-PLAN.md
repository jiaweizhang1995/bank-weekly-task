## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user tap -> export handler | 高频点击和中断操作进入前端异步导出流程 |
| DOM capture -> generated blob | 页面内容被转换为图片，可能因节点状态或浏览器限制失败 |
| generated file -> Web Share API | 浏览器原生分享能力不可用、拒绝或抛错 |
